Realice un Formulario en HTML  que permita escribir
Nombre
Coreo Electrónico 
Mensaje

Utilizando los tipos de campo Texto para el nombre , e mail para el correo  y textarea para escribir el mensaje 

UTILIZAR Botones SUBMIT y RESET para enviar y Borrar. Utilizar Hoja de estilo para el formulario y darle un diseño apropiado
La Etiqueta FORM ( <FORM> ) Es la que crea el formulario 




<form action="/my-handling-form-page" method="post"> 
  <ul>
<li>

<label for="name">Nombre:</label> 
  <input type="text" id="name" name="user_name" />

</li>
<li>
<label for="mail">Correo electrónico:</label> 

<input type="email" id="mail" name="user_mail" />
</li>

<li>
<label for="msg">Mensaje:</label>
<textarea id="msg" name="user_message"></textarea>
</li> 
  </ul>

</form>
